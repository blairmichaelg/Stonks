## Packages
recharts | For visualizing equity curves and performance metrics
react-hook-form | For robust form handling in the strategy builder
@hookform/resolvers | For Zod validation integration with forms
framer-motion | For smooth transitions between dashboard states
clsx | For conditional class merging
tailwind-merge | For handling Tailwind class conflicts

## Notes
- The app uses a "Fintech" dark theme by default.
- Recharts will be used for the Equity Curve visualization on the details page.
- Strategy parsing is an async operation that returns a JSON structure preview before saving.
